# 🚀 Inside A Computer — A Digital Odyssey

An interactive web experience that visually explores the internal architecture of a modern computer system.
This project transforms complex hardware concepts into an immersive, animated journey.

---

## 🌐 Live Demo

🔗 https://inside-a-computer.netlify.app/

---

## 📌 Features

* 🎯 Interactive sections: CPU, RAM, GPU, Storage, Network
* 🎨 Advanced UI with neon-themed design
* 🧠 Typewriter animations for dynamic text
* 🖱️ Custom cursor and hover interactions
* ⚙️ Data Bus simulation (real-time concept visualization)
* 🔊 Ambient audio system
* 🌌 Three.js powered 3D graphics
* 📱 Responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Advanced animations & effects)
* JavaScript (Vanilla JS)
* Three.js (3D rendering)
